🐾 VidaPet

Plataforma desenvolvida para auxiliar na adoção e cuidado de animais, permitindo o gerenciamento de pets, usuários e informações relacionadas ao processo de adoção.

---

📌 Sobre o Projeto

O VidaPet foi criado com o objetivo de conectar pessoas interessadas em adoção responsável de animais, oferecendo uma plataforma simples, organizada e intuitiva.

O sistema conta com funcionalidades voltadas para cadastro de animais, gerenciamento de informações e interação entre usuários.

---

🚀 Tecnologias Utilizadas

## Front-End

* HTML5
* CSS3
* JavaScript

## Back-End

* Python

## Banco de Dados

* SQL

---

# ✨ Funcionalidades

* Cadastro de animais
* Exibição de pets disponíveis
* Sistema de comentários
* Gerenciamento de informações
* Integração com banco de dados SQL

# 💡 Melhorias Futuras

* Sistema de login

* Favoritar animais

# 🎯 Objetivo do Projeto

O objetivo do VidaPet é utilizar tecnologia para ajudar animais e incentivar a adoção responsável, proporcionando uma experiência simples e acessível para os usuários.
